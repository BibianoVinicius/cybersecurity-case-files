# SentinelX — Cyber Awareness Platform

Projeto desenvolvido utilizando IA Generativa no PartyRock AWS com foco em conscientização em segurança digital e simulação de funcionalidades SOC.

---

## 🔥 Funcionalidades

- Análise de phishing
- Simulação de alertas SOC
- Geração de senhas seguras
- Quiz de cybersecurity
- Security Awareness
